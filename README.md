# 🚀 Quorbit Agency Website

A modern fullstack digital marketing agency website built using React, Node.js, Express, and MySQL.

---

# 🌐 Features

- Premium responsive UI
- Responsive mobile navbar
- Modern landing page
- Contact consultation form
- MySQL database integration
- Node.js backend API
- Toast notifications
- Loading states
- Service pages
- Fullstack architecture

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Axios
- React Hot Toast
- React Icons

## Backend
- Node.js
- Express.js
- MySQL
- dotenv
- CORS
